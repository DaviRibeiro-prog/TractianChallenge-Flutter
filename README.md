SOME FEATURES OF THE CHALLENGE:

- Complexly filtered lists by priority.
- QRcode.
- Date manipulation with timestamp.
- Front and back interaction.
- Model manipulation.
- Design difficulties - Dynamic Checkbox, Dialog with Listview...).
- Responsive line chart.


LIBRARIES USED:

- Modular (Dependency injection and navigation).
- Dio (Http, API Interaction).
- Intl (Date Formatting Timestamp).
- Syncfusion (For the line chart).
- Path provider and Cached Image.
- Dropdownbellow.
- QRcode Scanner.

OBSERVATION:

Folders separated by features (Main, Work Orders, Assets and the QRcode).
Models separated by features as well (Work Orders and Assets).
I used MODULAR, MVC pattern with the abstraction of all repositories.
Contains an 'utils' folder, with global components, singletons and design classes
